## Input/Output in C
**Output:**
- Character output
	- `putchar`
	- `puts`
	- `fputs`
	- `fputc`
- Formatted output
	- `printf`
	- `fprintf`
**Input:**
- Character input
	- `getchar`
	- `gets`
	- `fgets`
	- `fgetc`
- Formatted input 
	- `scanf`
	- `fscanf`
