## Модули math и numpy
### Библиотека math
#### Функции теории чисел и функции представления.

 - `math.comb(n, k)` - возвращает количество сочетаний из `n` элементов по `k` элементам без повторений и без учета порядка.

```python
import math

print(math.comb(12, 3)) # 220
```

- `math.factorial(x)` - возвращает факториал целого неотрицательного числа `x`.

```python
print(math.factorial(5)) # 120
```

- `math.gcd(*integers)` - возращает наибольший общий делитель (НОД) для чисел-аргументов.

```python
print(math.gcd(120, 210, 360)) # 30
```

- `math.lcm(*integers)` - возвращает наименьшее общее кратное (НОК) для чисел-аргументов.

```python
print(math.lcm(10, 20, 30, 40)) # 120
```

- `math.perm(n, k=None)` - возвращает количество размещений из `n` элементов по `k` элементам без повторений и с учетом порядка. Если значение аргумента `k` не задано, то возвращается количество перестановок множества из `n` элементов.

```python
print(math.perm(4, 2)) # 12
print(math.perm(4)) # 24
```

- `math.prod(iterable, start=1)` - возвращает произведение элементов итерируемого объекта `iterable`. Если `iterable` пустой, то возращается значение именованного аргумента `start`.

```python
print(math.prod(range(12, 21))) # 6704425728000
```

#### Степенные и логарифмические функции.

- `math.exp(x)` - возращает значение экспоненциальной функции $e^x$.

```python
print(math.exp(3.5)) # 33.11545195869231
```

- `math.log(x, base)` - возращает значение логарифма от `x` по основанию `base`. Если значение аргумента `base` не задано, то вычисляется натуральный логарифм. Вычисление производитеся по формуле `log(x) / log(base)`.

```python
print(math.log(10)) # 2.302585092994046 2
print(math.log(10, 2)) # 3.3219280948873626
```

- `math.pow(x, y)` - возращает значение `x` в степени `y`. В отличие от операции `**`, происходит преобразование обоих аргументов в вещественные числа.

```python
print(math.pow(2, 10)) # 1024.0
print(math.pow(4.5, 3.7)) # 261.1477575641718
```

#### Тригонометрические функции
Доступны функции синус (`sin(x)`), косинус (`cos(x)`), тангенс (`tan(x)`), арксинус (`asin(x)`), арккосинус (`acos(x)`), арктангенс (`atax(x)`). Обратите внимание: угол задается и возращается в радианах.

- `math.dist(p, q)` - возращает Евклидово расстояние между точками `p` и `q`, заданнами как итерируемые объекты одной длины.

```python
print(math.dist((0, 0, 0), (1, 1, 1))) # 1.7320508075688772
```

- `math.hypot(*coordinates)` - возращает длину многомерного вектора с координатами, заданным в позиционных аргументах `coordinates`, и началом в центры системы координат. 

```python
print(math.hypot(1, 1, 1)) # 1.7320508075688772
print(math.hypot(3, 4)) # 5.0
```

#### Функции преобразования угла

- `math.degrees(x)` - преобразует угол из радианов в градусы.

```python
print(round(math.degrees(math.asin(0.5)), 1)) # 30.0
```

- `math.radians(x)` - преобразует угол из градусов в радианы.

```python
print(round(math.sin(math.radians(30)), 1)) # 0.5
```
